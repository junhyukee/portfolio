Projects

"Skincare"

It's frustrating when our skin is not how we want it to be. "Skincare" is a web application that was made to pinpoint which ingredients in products may cause you to break out. We used Vue.js, Express + Node, and Google Cloud Datastore.

Company Site

A sample company site coded from scratch, using HTML5, CSS3, and Javascript. The site includes a fully functional tab component and a fully functional fade carousel, both coded from scratch utilizing Javascript components with the DOM.

Ski Saver

ACL tears are the most common injury among skiiers. Ski Saver is a device that determines whether a skiier will undergo an injury, utilizing an accelerometer, force sensors, and a motor. Ski Saver was created using an Arduino and integrated with LabVIEW.

Electric Guitar

A fun project where we started off at the woodshop to carve the wood for the guitar. We designed the circuit board with amplifiers and filters to create a fully functional electric guitar.
